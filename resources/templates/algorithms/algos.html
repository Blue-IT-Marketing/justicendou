Search Algorithms

Implementations of the classic depth first and breadth first algorithms in Javascript

Depth First Search (DFS)

Definition - Wikipedia = Depth-first search (DFS) is an algorithm for traversing or searching tree or graph data structures. The algorithm starts at the root node (selecting some arbitrary node as the root node in the case of a graph) and explores as far as possible along each branch before backtracking.

Breadth First Search (BFS)

Definition - on Wikidedia - Breadth-first search (BFS) is an algorithm for traversing or searching tree or graph data structures. It starts at the tree root (or some arbitrary node of a graph, sometimes referred to as a 'search key'[1]), and explores all of the neighbor nodes at the present depth prior to moving on to the nodes at the next depth level.

Neural Network Algorithms

Are algorithms which are loosely based on the way that a brain processes information through synaptic connections between neurons, algorithms created in this way allows for computer algorithms to come up with solutions to problems that traditional algorithms either find hard to solve or cannot yet solve.

Connection between Neural Networks and Data

Neural Networks depend entirely on the available information for training purposes and most often also in the pre processing of such information in order to organize it in such a way that allows for the information to be processed as inputs to the network.
The depence on information of neural networks is the sole reason why the re-emergence of neural networks today happened just after the revolution of Cloud Computing, and also (Big Data) Data Analytics and Processing Methods have matured.
